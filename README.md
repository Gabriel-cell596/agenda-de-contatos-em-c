# Agenda de Contatos em C

Trabalho final da disciplina de **Programação** do primeiro ano do curso de **Sistemas de Informação** na **UENP**.  

O programa é um sistema simples para gerenciamento de contatos, desenvolvido em linguagem C. Ele permite cadastrar, listar, buscar, editar e excluir contatos, salvando os dados em arquivo CSV.

---

## Funcionalidades

- **Cadastrar contato**: adiciona novos contatos com nome, telefone e email.  
- **Listar contatos**: exibe todos os contatos cadastrados em formato de tabela.  
- **Buscar contato**: permite procurar por nome, telefone ou email.  
- **Editar contato**: altera informações de contatos existentes usando o ID do contato.  
- **Excluir contato**: remove contatos da lista pelo ID.  
- **Salvar contatos**: todos os dados são salvos automaticamente em `contatos.csv`.
